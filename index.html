Pig Dice

Game Rules

First Player to 100 pts wins!

  • Player 1 goes first
  • Click the "Roll" button to roll a virtual 6-sided die
  • Values for 2-6 get added to the "Roll Total" value for each turn
  • If you roll a 1, your turn is over and you lose all points earned
  • To end turn, select "Hold" button- this will add the "Roll Total" to the player's score
